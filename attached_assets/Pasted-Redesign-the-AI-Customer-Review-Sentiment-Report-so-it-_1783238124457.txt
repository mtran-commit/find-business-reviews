Make the report look more like a premium modern proposal/editorial document:
- very clean
- easy to read
- strong layout structure
- lots of whitespace
- bold typography
- black / white / grey design
- only photos/images can be in colour

Important visual direction:
The report should feel inspired by the proposal example in terms of:
- page structure

Design rules:
1. Colour palette
- Use only:
  - black
  - white
  - light grey
  - medium grey
  - dark grey
- Do NOT use purple, green, orange, red, blue or multiple accent colours in the report layout.
- The only colour allowed should be inside business photos, logos, social profile pictures, or business images.
- All charts, dividers, labels, cards, and UI elements must be monochrome.

2. Overall style
- Minimal, modern, editorial, proposal-style layout
- Premium and structured
- Strong typography hierarchy
- Spacious and uncluttered
- Visually polished and professional
- Easy for business owners to scan quickly

3. Typography
- Use bold, large headings similar to the reference proposal
- Section titles should be oversized and striking
- Keep body text smaller, clean, and readable
- Use strong contrast between headings and paragraph text
- Use a modern sans-serif font
- Avoid overly decorative styles

4. Page structure
Use a structured proposal-style layout across the report:
- strong title section
- large headings
- clear content blocks
- image sections
- summary cards
- consistent alignment
- lots of breathing room

Take inspiration from the proposal example:
- top horizontal label/header strip
- large page title
- left or side block elements
- alternating text and image areas
- strong visual rhythm from page to page

- Add a clean top header bar on every page

- Add a simple footer with page number and report name
- Keep header and footer monochrome and minimal
