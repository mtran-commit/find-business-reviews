Add OpenAI AI review analysis logic to the Compare Reviews app.

Important security rule:
Do NOT put the OpenAI API key in the frontend HTML or JavaScript.
Create a backend API route that calls OpenAI securely using an environment variable:
OPENAI_API_KEY

The frontend should call our own backend route:
POST /api/analyze-reviews

The backend route should receive normalised review data from the frontend or from our Search API.

Backend task:
Use OpenAI to analyse the review data and return structured JSON only.

The AI should return:
{
  "trustScore": 0-100,
  "trustLabel": "Strong Reputation" | "Stable" | "Mixed" | "Check Carefully",
  "ratingGapAlert": "short warning or positive message",
  "reviewSummary": "short customer-friendly summary",
  "bestFor": ["tag1", "tag2", "tag3", "tag4"],
  "redFlags": ["flag1", "flag2"],
  "reputationTrend": "short trend explanation",
  "recommendedAction": "short advice for the user",
  "confidence": "High" | "Medium" | "Low"
}

Prompt rules:
- Keep language simple and useful for consumers.
- Do not claim the business is definitely good or bad.
- Use words like “appears”, “suggests”, “may”, “based on available review data”.
- Mention if data is limited.
- If rating gap is high, warn the user to read recent reviews.
- If review count is low, mention low confidence.
- Do not invent facts that are not in the input.
- Return JSON only. No markdown.

Create this backend in Node.js/Express.

Install dependency:
npm install openai express cors dotenv

Create server.js with:
- express
- cors
- dotenv
- OpenAI client
- POST /api/analyze-reviews route
- validation for missing businessName or ratings
- try/catch error handling
- return 500 error if OpenAI fails

Use this model:
gpt-4.1-mini

Use OpenAI Responses API.

Example server.js:

const express = require("express");
const cors = require("cors");
const dotenv = require("dotenv");
const OpenAI = require("openai");

dotenv.config();

const app = express();
app.use(cors());
app.use(express.json({ limit: "1mb" }));

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

app.post("/api/analyze-reviews", async (req, res) => {
  try {
    const { businessName, address, ratings } = req.body;

    if (!businessName || !ratings) {
      return res.status(400).json({
        error: "Missing businessName or ratings"
      });
    }

    const prompt = `
You are the AI review analyst for Compare Reviews.

Analyse this business review data and return JSON only.

Business name:
${businessName}

Address:
${address || "Not provided"}

Review data:
${JSON.stringify(ratings, null, 2)}

Return this exact JSON structure:
{
  "trustScore": 0,
  "trustLabel": "",
  "ratingGapAlert": "",
  "reviewSummary": "",
  "bestFor": [],
  "redFlags": [],
  "reputationTrend": "",
  "recommendedAction": "",
  "confidence": ""
}

Rules:
- trustScore must be a number between 0 and 100.
- trustLabel must be one of: Strong Reputation, Stable, Mixed, Check Carefully.
- bestFor must contain 2 to 5 short tags.
- redFlags must contain 0 to 4 short warnings.
- confidence must be High, Medium, or Low.
- Be careful and balanced.
- Do not invent facts.
- Return JSON only.
`;

    const response = await openai.responses.create({
      model: "gpt-4.1-mini",
      input: prompt
    });

    const rawText = response.output_text;

    let aiResult;

    try {
      aiResult = JSON.parse(rawText);
    } catch (parseError) {
      return res.status(500).json({
        error: "OpenAI returned invalid JSON",
        raw: rawText
      });
    }

    return res.json(aiResult);
  } catch (error) {
    console.error("OpenAI analysis error:", error);

    return res.status(500).json({
      error: "Failed to analyse reviews"
    });
  }
});

const PORT = process.env.PORT || 3000;

app.listen(PORT, () => {
  console.log(`Compare Reviews server running on port ${PORT}`);
});
