Before sending the user to Stripe for the Business Reputation Report, collect report request details in our app.

Current problem:
The button goes straight to Stripe, but we need to collect:
- customer full name
- email address
- phone number
- business name
- business address
- business website or Google profile link

New flow:
1. User clicks “Unlock full report — $10”
2. Open a modal called:
   Business Reputation Report Request
3. Show text:
   “Enter your details so we can prepare and send your AI Business Reputation Report after payment.”
4. Display form fields:
   - Full name
   - Email address
   - Phone number
   - Business name
   - Business address
   - Business website or Google profile link
5. Add checkbox:
   “I confirm I am the business owner, manager, or authorised representative.”
6. Add button:
   “Continue to secure payment”
7. Validate the form before payment.
8. Save the request to the backend/database as pending_payment.
9. Then open the Stripe Payment Link:
   https://buy.stripe.com/5kQ14oa2yfUh9JR1Tt2cg00

Do not collect credit card details inside our app.
Stripe handles payment securely.

async function submitReportRequest(event) {
  event.preventDefault();

  const form = event.target;
  const formData = new FormData(form);

  const payload = {
    fullName: formData.get("fullName"),
    email: formData.get("email"),
    phone: formData.get("phone"),
    businessName: formData.get("businessName"),
    businessAddress: formData.get("businessAddress"),
    businessLink: formData.get("businessLink"),
    authorised: formData.get("authorised") === "on",
    searchedBusiness: currentData?.business || null,
    status: "pending_payment",
    createdAt: new Date().toISOString()
  };

  if (!payload.authorised) {
    alert("Please confirm you are authorised to request this report.");
    return;
  }

  try {
    const response = await fetch("/api/report-requests", {
      method: "POST",
      headers: {
        "Content-Type": "application/json"
      },
      body: JSON.stringify(payload)
    });

    const data = await response.json();

    if (!response.ok) {
      throw new Error(data.error || "Could not save request");
    }

    window.open("https://buy.stripe.com/5kQ14oa2yfUh9JR1Tt2cg00", "_blank");
  } catch (error) {
    alert("Could not save your report request. Please try again.");
  }
}

Backend:
Add this route:

app.post("/api/report-requests", async (req, res) => {
  try {
    const {
      fullName,
      email,
      phone,
      businessName,
      businessAddress,
      businessLink,
      authorised,
      searchedBusiness,
      status
    } = req.body;

    if (!fullName || !email || !phone || !businessName || !businessAddress || !authorised) {
      return res.status(400).json({
        error: "Missing required report request details"
      });
    }

    const request = {
      id: "rr_" + Date.now(),
      fullName,
      email,
      phone,
      businessName,
      businessAddress,
      businessLink,
      authorised,
      searchedBusiness,
      status: status || "pending_payment",
      createdAt: new Date().toISOString()
    };

    // For now, save to memory or a JSON file.
    // Later, save this to the real database.

    if (!global.reportRequests) {
      global.reportRequests = [];
    }

    global.reportRequests.push(request);

    return res.json({
      success: true,
      requestId: request.id
    });
  } catch (error) {
    console.error("Report request error:", error);
    return res.status(500).json({
      error: "Could not create report request"
    });
  }
});
